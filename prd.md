# CityStoryMap — Product Requirements Document

> 도시를 테마별 이야기로 탐험하는 감성 모바일 앱

---

## 1. 제품 개요

**CityStoryMap**은 특정 도시를 배경으로 한 소설·역사·애니메이션 등의 테마를 골라, 그 이야기 속 실제 장소들을 지도와 카드로 탐험하는 앱이다.

- "교토에 가기 전, 미시마 유키오의 금각사 루트를 먼저 읽어보자"
- "사카모토 료마의 발자취를 따라 교토를 걷고 싶다"

단순한 관광 정보가 아니라, **이야기가 있는 장소들**을 큐레이션하는 문학·문화 여행 가이드.

---

## 2. 핵심 개념

| 개념 | 설명 |
|------|------|
| **도시 (City)** | 앱의 최상위 단위. 예: 교토, 도쿄 |
| **테마 (Theme)** | 하나의 이야기 루트. 소설/애니메이션/역사 인물 등 하나의 테마가 여러 장소를 묶음 |
| **장소 (Place)** | 테마에 속한 실제 방문 가능 장소. 이야기 맥락과 함께 제공 |

---

## 3. 콘텐츠 데이터 모델

### 도시 (City)
- `name_ko` / `name_en` (예: 교토 / Kyoto)
- `description` — 짧은 소개문

### 테마 (Theme)
- `title` — 테마 이름 (예: "미시마 유키오의 금각사 루트")
- `category` — 소설 / 애니메이션 / 역사 / 영화 등
- `year` — 작품/사건 연도 (예: 1956년)
- `hook_text` — 첫 화면에 표시되는 훅 문구 (1~2문장)
- `hero_gradient` — 히어로 배경 그라디언트 색상
- `place_count` — 연결된 장소 수
- `is_featured` — 캐러셀 노출 여부 (운영자 설정)
- `featured_order` — 캐러셀 순서 (운영자 설정)

### 장소 (Place)
- `name` — 장소명 (예: 금각사(킨카쿠지))
- `emoji` — 대표 이모지 (이미지 대체용)
- `district` — 구/지역 (예: 기타야마 · 교토)
- `story_quote` — 이 장소에 얽힌 이야기 발췌/설명 (2~3문장, 이탤릭체)
- `tags[]` — 카테고리 태그 (예: ["1950년대", "사원"])
- `order` — 테마 내 순서
- `lat` / `lng` — 지도 좌표

### 관련작품 (RelatedWork)
- `id` — 고유 식별자
- `type` — 작품 유형 (book | movie | drama | anime)
- `title` — 작품명 (예: "燃えよ剣")
- `creator` — 저자/감독 (예: "시바 료타로")
- `year` — 출시연도 (예: 1962)
- `cover_emoji` — 표지 대신 이모지 (MVP용, 예: 📖, 🎬)
- `external_url` — 외부 링크 (아마존, 넷플릭스 등)
- `theme_ids[]` — 연결된 테마 ID 목록

## 4. 화면 구성

### 4-1. 홈 화면 (도시별 스토리 목록)

**확정 레이아웃: 캐러셀형**

```
┌─────────────────────┐
│  CityStoryMap   교토 │  ← 상단 네비
├─────────────────────┤
│                     │
│   [자동 슬라이드     │  ← 히어로 캐러셀 (220px)
│    히어로 카드]      │    - 3.5초 자동 전환
│    ● ○ ○ ○          │    - 하단 닷 인디케이터
├─────────────────────┤
│ 교토의 모든 이야기  12│  ← 섹션 헤더 + 총 개수
├─────────────────────┤
│ 📖 미시마 유키오...  ›│
│ 🌌 〈언어의 정원〉... ›│  ← 스토리 리스트
│ ⛩️ 신센구미가...     ›│    (무한 스크롤 또는 전체 표시)
│ ...                  │
├─────────────────────┤
│ 📖 스토리 🗺️ 지도 📚 라이브러리│  ← 하단 탭 바
└─────────────────────┘
```

**캐러셀 슬라이드 구성 요소:**
- 배경: 테마별 어두운 그라디언트
- 카테고리 + 연도 (예: "소설 · 1956년")
- 테마 제목 (이탤릭, 흰색)
- 훅 문구 (옅은 흰색)
- 장소 수 + "탐험하기 →" CTA

**스토리 리스트 행:**
- 이모지 썸네일 (40×40, 둥근 모서리)
- 테마 제목
- 카테고리 + 장소 수 (예: "소설 · 7곳")
- 화살표 `›`

**운영 정책:**
- 캐러셀 슬라이드: 운영자가 순서와 노출 여부 관리
- 새 스토리 추가 시 자동으로 리스트에 반영

### 4-2. 테마 상세 화면

**확정 디자인: v3 (밝은 톤 + 풍부한 카드 이미지)**

```
┌─────────────────────┐
│ ← 교토          7곳  │  ← 히어로 헤더 (130px, 어두운 그라디언트)
│ 미시마 유키오의      │
│ 금각사 루트          │
├────────┬────────┬───────┤
│ 카드뷰 │ 지도뷰 │관련작품│  ← 뷰 전환 탭 (3탭)
├─────────────────────┤
│ ┌───────────────┐   │
│ │ 01    🏯      │   │  ← 장소 카드
│ │ 금각사(킨카쿠지)│  │    이미지 영역: 120px
│ │       기타야마 │   │    - 어두운 그라디언트 배경
│ └───────────────┘   │    - 이모지 (42px)
│ "미조구치가 집착한..."│    - 장소명 + 지역 오버레이
│ [1950년대] [사원] →  │  ← 스토리 인용문 + 태그 + CTA
│ ┌───────────────┐   │
│ │ 02    ⛩️      │   │
│ ...                  │
├─────────────────────┤
│ 📖 스토리 🗺️ 지도 📚 라이브러리│
└─────────────────────┘
```

**카드 이미지 영역 구성:**
- 배경: 각 장소별 어두운 그라디언트 (테마 컬러 계열)
- 텍스처 오버레이 패턴
- 순번 (01, 02...)
- 이모지 (중앙, drop-shadow)
- 장소명 (이탤릭, 흰색) — 하단 좌측
- 지역명 — 하단 우측

**카드 본문 구성:**
- 이야기 인용문/설명 (이탤릭, 2~3줄)
- 태그 (pill 형태)
- "보기 →" CTA → 장소 상세 화면으로 이동

### 4-2-1. 장소 상세 화면

테마 상세 화면에서 장소 카드를 터치하면 이동하는 화면.
해당 장소의 상세 스토리와 방문 정보를 제공.

```
┌─────────────────────┐
│ ←                   │  ← SliverAppBar (스크롤 시 축소)
│       🏯            │     - 테마 그라디언트 배경
│                     │     - 이모지 (48px)
│ 금각사(킨카쿠지)     │     - 장소명 (이탤릭, 흰색)
│ 기타야마             │     - 지역명
├─────────────────────┤
│ [1950년대] [사원]    │  ← 태그들
├─────────────────────┤
│ ❝                   │  ← 인용문 카드
│ 미조구치가 집착한    │     (storyQuote)
│ "절대적 아름다움"... │
├─────────────────────┤
│ 📖 이야기           │  ← 상세 스토리 섹션
│ 1950년, 금각사는... │     (detailStory)
├─────────────────────┤
│ 👤 관련 인물         │  ← 관련 인물 섹션
│ 미시마 유키오는...   │     (relatedPerson)
├─────────────────────┤
│ 💡 방문 팁          │  ← 방문 팁 섹션 (배경색 다름)
│ 오전 일찍 방문하면...│     (visitTip)
├─────────────────────┤
│ ┌─────────────────┐ │  ← 지도 미리보기 (목업)
│ │    🏯           │ │     - 좌표 표시
│ │  35.0394, 135.7 │ │     - Google Maps 연동 예정
│ └─────────────────┘ │
└─────────────────────┘
```

**장소 상세 화면 구성 요소:**
- 히어로 헤더: 테마 그라디언트 배경 + 이모지 + 장소명 + 지역
- 태그: pill 형태로 나열
- 인용문 카드: 짧은 스토리 인용 (storyQuote)
- 상세 이야기: 장소에 얽힌 풍부한 스토리 (detailStory)
- 관련 인물: 해당 장소와 관련된 역사적/문학적 인물 (relatedPerson)
- 방문 팁: 실제 방문 시 유용한 정보 (visitTip)
- 지도 미리보기: 좌표 표시 (추후 Google Maps 연동)

### 4-4. 글로벌 지도 탭 (홈 하단 탭 → 지도)

- 해당 도시 전체 지도
- 모든 테마의 핀이 표시됨
- 핀 클릭 → 테마 상세로 이동

---

### 4-5. 라이브러리 탭 (홈 하단 탭 → 라이브러리)

도시와 관련된 모든 작품(책, 영화, 드라마, 애니메이션)을 탐색하는 화면.
테마를 선택하지 않아도 접근 가능하며, 여행 전/후 콘텐츠 탐색 용도.

```
┌─────────────────────┐
│  📚 라이브러리       │  ← 상단 헤더
├─────────────────────┤
│ 교토를 더 깊이       │
│ 만나는 작품들        │  ← 섹션 타이틀
├─────────────────────┤
│[전체][소설][영화]... │  ← 카테고리 필터 (가로 스크롤)
├─────────────────────┤
│ 📖 燃えよ剣          │
│ 시바 료타로 · 1962   │  ← 작품 카드
│ → 신선조 협객        │    연결된 테마 표시
│ [아마존에서 보기 →]   │    외부 링크 CTA
├─────────────────────┤
│ 🎬 壬生義士伝        │
│ 타키타 요지로 · 2003 │
│ → 신선조 협객        │
│ [넷플릭스에서 보기 →] │
├─────────────────────┤
│ ...                  │
├─────────────────────┤
│ 📖 스토리 🗺️ 지도 📚 라이브러리│
└─────────────────────┘
```

**작품 카드 구성:**
- 커버 이모지 (MVP: 실제 이미지 대신 📖🎬📺🎞️ 사용)
- 작품명 (굵게)
- 저자/감독 · 연도
- 연결된 테마명 (클릭 시 테마 상세로 이동)
- 외부 링크 CTA (클릭 시 외부 URL 오픈)

**카테고리 필터:**
- 전체 / 소설 / 영화 / 드라마 / 애니메이션
- 선택 시 해당 타입만 필터링

### 4-6. 테마 상세 내 관련작품 탭

테마 상세 화면의 세 번째 탭. 해당 테마와 연결된 작품만 표시.

```
┌─────────────────────┐
│ ← 교토          7곳  │
│ 신선조 협객의 교토   │
├────────┬────────┬───────┤
│ 카드뷰 │ 지도뷰 │관련작품│  ← 관련작품 탭 선택됨
├─────────────────────┤
│ 📚 이 이야기를 더    │
│    깊이 만나보세요   │
├─────────────────────┤
│ 📖 燃えよ剣          │
│ 시바 료타로 · 1962   │
│ [아마존에서 보기 →]   │
├─────────────────────┤
│ 🎬 壬生義士伝        │
│ 타키타 요지로 · 2003 │
│ [넷플릭스에서 보기 →] │
├─────────────────────┤
│ 📖 스토리 🗺️ 지도 📚 라이브러리│
└─────────────────────┘
```

**라이브러리 탭과의 차이점:**
- 라이브러리: 도시 전체 작품 (카테고리 필터 있음)
- 테마 내 관련작품: 해당 테마 연결 작품만 (필터 없음, 간결)

## 5. 디자인 시스템

### 컬러
| 역할 | 값 |
|------|----|
| 배경 (기본) | `#faf9f6` |
| 배경 (페이지) | `#f5f3ee` / `#edeae3` |
| 텍스트 (기본) | `#2c2c2c` |
| 텍스트 (보조) | `#888`, `#bbb` |
| 액센트 (브라운골드) | `#8b6f4e` |
| 구분선 | `#ede9e3` |
| 히어로 배경 | 테마별 어두운 그라디언트 |

### 타이포그래피
- 제목/본문: `Georgia` (serif) — 감성·문학적 톤
- UI 레이블/메타: `sans-serif` (system)
- 이탤릭체: 테마 제목, 장소명, 인용문에 사용

### 전반적 톤
- 잡지·문학 감성
- 어둡고 극적인 히어로 vs 밝고 따뜻한 본문 대비
- 과도한 컬러 사용 지양 — 절제된 earth tone
